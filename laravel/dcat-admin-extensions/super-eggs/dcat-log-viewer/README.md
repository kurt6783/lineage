# Dcat Log Viewer

## 介绍

为 Dcat Admin 添加在后台查看系统日志的功能。

> 该扩展包是基于 dcat 2.0 开发。dcat 1.0 暂不可用，后续开发。

![WX20210310-140858@2x](https://i.loli.net/2021/03/10/mBKvlSi59DJVu6f.png)

## 安装

```bash
composer require super-eggs/dcat-log-viewer
```
